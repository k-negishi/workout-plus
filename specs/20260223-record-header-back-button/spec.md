# 仕様書: RecordScreen ヘッダー改修（Issue #131）

## 概要

RecordScreen 上部のヘッダーに戻るボタンを追加し、日付タイトルを中央寄せ、青系統のカラーに変更する。

## ユーザーストーリー

### US-1: 戻るボタンの追加
**ユーザーとして、** RecordScreen の上部左端に戻るボタンを表示し、
タップすると前の画面（HomeScreen または CalendarScreen）に戻れるようにしたい。
**なぜなら、** 現在はジェスチャー（スワイプバック）でしか戻れないため、
iOS/Android 両方で明示的な操作で戻れる経路が必要だから。

### US-2: 日付タイトルの中央寄せ
**ユーザーとして、** ヘッダーの「XX月XX日のワークアウト」テキストが中央に配置されている状態で
どの日のワークアウトを記録しているか一目で確認したい。
**なぜなら、** 現在は左寄せ・小さいサイズで目立たないから。

### US-3: 青系統カラーへの変更
**ユーザーとして、** RecordScreen のヘッダー（背景・タイトル文字・戻るボタン）が
青系統のカラーで統一されていることで、記録中であることを視覚的に認識したい。
**なぜなら、** 白背景だと通常画面との区別がつきにくいから。

## 機能仕様

### F-01: ヘッダーレイアウト

ヘッダーは3列構成（left / center / right）で実装する。

```
┌──────────────────────────────────────────────┐
│ [←]        2月23日のワークアウト             │
└──────────────────────────────────────────────┘
```

- **背景**: `#4D94FF`（メインカラー）
- **高さ**: `paddingVertical: 12` + `insets.top` 分の paddingTop
- **左エリア**: 戻るボタン（Ionicons `chevron-back`、サイズ 24、色 `#ffffff`）
- **中央エリア**: 日付テキスト（`M月d日のワークアウト` フォーマット、`16px`、`600`、`#ffffff`、`text-center`）
- **右エリア**: 空（左ボタンとバランスを取るためのスペーサー）

### F-02: 戻るボタン動作

- `navigation.goBack()` を呼び出す
- HomeStack 内・CalendarStack 内どちらでも動作する
- `accessibilityLabel="戻る"` を設定する

### F-03: 既存の白いヘッダー View を置き換える

現在の白いヘッダー View（`backgroundColor: '#ffffff'`、左寄せテキスト）を
F-01 の新しいヘッダーに置き換える。

## 非機能要件

- SafeArea 対応: `insets.top` を paddingTop に適用（現行実装を維持）
- コンテナ背景色 `#f9fafb` は維持（ヘッダー以外は変更しない）
- `record-screen-container` の `testID` は維持（既存テストとの互換性）
