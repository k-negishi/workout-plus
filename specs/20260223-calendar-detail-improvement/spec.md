# Issue #133: カレンダー詳細コンポーネント改善

## 概要
DaySummary コンポーネントの表示情報を拡充し、UX を向上させる。

## 要件

### 1. set 文言追加
- 各セット行のセット番号表示に `set` を付与する
- 変更前: `1  100kg × 10`
- 変更後: `1 set  100kg × 10`

### 2. ワークアウトメモ・種目メモの表示
- **ワークアウトメモ**: workout.memo が存在する場合、サマリーカードの下に表示
- **種目メモ**: workout_exercise.memo が存在する場合、各種目カード内のセット一覧の下に表示
- メモがない場合は何も表示しない（空欄を確保しない）

### 3. 既に実装済みの機能（変更不要）
- 1RM 換算表示（各セット右端に表示済み）
- セット詳細表示（各セットの weight/reps 表示済み）
- ワークアウト削除ボタン（DaySummary 下部に表示済み）
- 削除確認ダイアログ（CalendarScreen で ConfirmDialog 使用済み）

## 影響範囲
- `DaySummary.tsx`: set 文言追加、メモ表示
- `DaySummary.test.tsx`: 新規テスト追加

## デザイン仕様
- メモテキスト: fontSize 13, color #64748b（secondary text）
- メモアイコン: なし（テキストのみ）
- メモ領域: paddingTop 8, 種目カード内の下部に配置
