# エクササイズアイコン選定

> **タイミング**: Phase 1 Flutter実装前（Task 9 デザインレビューまでに）
> **目的**: 種目別アイコンを既存ライブラリから統一スタイルで調達する

## 背景

- UIアイコン（ホーム、カレンダー等）は Lucide 風ストロークアイコンで問題なし
- エクササイズアイコン（ベンチプレス、スクワット等）は汎用ライブラリに存在しない
- 手描き SVG は 20px で判読困難。専用アイコンセットから取るのが業界標準

## 必要なアイコン（5種目）

| 部位 | 種目 | 用途 |
|---|---|---|
| 胸 | ベンチプレス | ワークアウトカード、種目選択 |
| 二頭 | アームカール | ワークアウトカード、種目選択 |
| 三頭 | ディップス | ワークアウトカード、種目選択 |
| 背中 | チンニング | ワークアウトカード、種目選択 |
| 脚 | スクワット | ワークアウトカード、種目選択 |

## 候補ソース

| ソース | URL | ライセンス | 備考 |
|---|---|---|---|
| Noun Project | https://thenounproject.com/browse/icons/term/bench-press/ | 無料(要帰属)/Pro $3/月 | 同一アーティストで揃えればテイスト統一可 |

| SVG Repo | https://www.svgrepo.com/vectors/gym/ | MIT/CC0が多い | そのままDL可 |

## 選定基準

- [ ] 5種目すべてが同一パック/アーティストに揃っているか
- [ ] ストロークスタイルで、既存UIアイコン（Lucide風）と統一感があるか
- [ ] 20px〜40px で判別できるか
- [ ] ライセンスが商用利用可か（帰属表示の有無）
- [ ] SVG形式でダウンロード可か

## 現状

WFフェーズでは手描きSVGを仮置き中（`requirements/adopted/icons/` 内）。
Flutter実装時に正式なアイコンセットに差し替える。
